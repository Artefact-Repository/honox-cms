一个高级组件，将 Table 与 Pagination 组合在一起，并在内部管理状态。

## 用法

<DemoPreview>
  <div style={{ width: "100%" }}>
    <PaginatedTable title="Users" pageSize={5} />
  </div>
</DemoPreview>

### CMS 页面构建器

该组件可作为 `paginatedTable` 区块在 [页面构建器](/docs/PageBuilder)（`content/pages/*.json`）中使用：

```json
{
  "type": "paginatedTable",
  "title": "Users",
  "pageSize": 5
}
```

## 属性

| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `title` | `string` | - | 表格标题。 |
| `pageSize` | `number` | `10` | 每页条数。 |

## CMS 配置

```yaml
- label: PaginatedTable
  name: paginated-table
  widget: object
  fields:
    - { label: Title, name: title, widget: string, required: false }
    - { label: Page Size, name: pageSize, widget: number, default: 10 }
```

## 水合

第 2 级：始终作为岛屿水合，以管理内部分页状态。
