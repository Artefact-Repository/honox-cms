Um componente que permite aos usuários inserir várias tags como uma lista.

## Uso

<DemoPreview>
  <div style={{ width: "100%", maxWidth: "320px" }}>
    <TagsInput label="Skills" defaultValue={["React", "Hono"]} />
  </div>
</DemoPreview>

### Construtor de páginas CMS

Este componente está disponível como um bloco `tagsInput` no [Construtor de páginas](/docs/PageBuilder) (`content/pages/*.json`):

```json
{
  "type": "tagsInput",
  "label": "Amenities",
  "defaultValue": ["WiFi", "Fireplace", "Hot Tub"]
}
```

## Propriedades

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `string[]` | - | Tags atuais. |
| `defaultValue` | `string[]` | - | Tags iniciais. |

## Configuração do CMS

```yaml
- label: TagsInput
  name: tagsInput
  widget: object
  fields:
    - { label: Label, name: label, widget: string, required: false }
    - { label: Default Tags, name: defaultValue, widget: list, required: false }
```

## Hidratação

Nível 2: Hidrata quando `value`, `defaultValue`, `onValueChange`, ou o estado de entrada está presente.
